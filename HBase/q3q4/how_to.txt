To init a HBase server:
    hbase shell
    Create table: create 'q3', {NAME => 'by', VERSIONS => 1}
    Create table: create 'q4', {NAME => 'tag', VERSIONS => 1}
    Start thrift: hbase-daemon.sh start thrift
